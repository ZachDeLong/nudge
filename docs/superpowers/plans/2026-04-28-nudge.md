# Nudge Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build Nudge, a macOS menu bar app that surfaces Claude Code permission prompts in a clickable popover so the user can approve from anywhere without alt-tabbing back to the terminal.

**Architecture:** Two-piece system. (1) `Nudge.app` — a SwiftUI menu bar app (LSUIElement) running an HTTP server on `127.0.0.1` that displays prompts in a popover anchored to its menu bar icon. (2) `nudge-hook` — a small Swift CLI binary wired into Claude Code's `PreToolUse` hook; reads JSON from stdin, POSTs to the app, blocks until the user clicks, writes the decision JSON to stdout. Both targets live in a single SwiftPM package.

**Tech Stack:** Swift 5.10+, SwiftUI, AppKit (`NSStatusItem`, `NSPopover`, `NSVisualEffectView`), `Network.framework` (`NWListener` for the TCP socket), in-house minimal HTTP/1.1 parser, Swift actors and `async/await` for concurrency. SwiftPM build (no Xcode required). macOS 14+.

**Responsibility per file:**

- `NudgeApp.swift` — process entry, wires everything up, owns the singleton `MenuBarController` and `PromptServer`.
- `MenuBarController.swift` — owns `NSStatusItem` + `NSPopover`. Knows nothing about HTTP. Subscribes to `PromptQueue` for the head-of-queue prompt and updates icon state.
- `PopoverView.swift` — pure SwiftUI view rendering a `Prompt` and emitting a `Decision`. Stateless w.r.t. the queue.
- `PromptServer.swift` — accepts TCP, hands raw bytes to `HTTPCodec`, hands parsed prompts to `PromptQueue`, suspends responses until the queue resolves them.
- `HTTPCodec.swift` — pure functions for parsing one HTTP/1.1 request and serializing one response. No I/O.
- `PromptQueue.swift` — actor. Enqueue prompts, suspend their futures, resolve them with a `Decision`. Notifies a listener (SwiftUI) when the head changes.
- `Prompt.swift` — `struct Prompt` (id, tool, command, cwd, sessionId), `enum Decision` (allow, deny).
- `PortFile.swift` — read/write `~/.config/nudge/port`.
- `NudgeHook/main.swift` — only file in the hook target. Reads stdin, locates Nudge, POSTs, prints stdout, falls back gracefully on any error.

```swift
// Sources/Nudge/HTTPCodec.swift
import Foundation

enum HTTPCodec {
    struct Request {
        let method: String
        let path: String
        let headers: [String: String]
        let body: [UInt8]
    }

    enum ParseError: Error {
        case malformed
        case needMoreData
    }

    /// Parses a full HTTP/1.1 request from `bytes`.
    /// Throws `.needMoreData` if the buffer is incomplete.
    static func parseRequest(_ bytes: [UInt8]) throws -> Request {
        // Find header/body separator: "\r\n\r\n"
        guard let headerEnd = findSequence([0x0D, 0x0A, 0x0D, 0x0A], in: bytes) else {
            throw ParseError.needMoreData
        }
        let headerBytes = Array(bytes[..<headerEnd])
        guard let headerString = String(bytes: headerBytes, encoding: .utf8) else {
            throw ParseError.malformed
        }
        let lines = headerString.components(separatedBy: "\r\n")
        guard let firstLine = lines.first else { throw ParseError.malformed }
        let parts = firstLine.split(separator: " ", maxSplits: 2, omittingEmptySubsequences: false).map(String.init)
        guard parts.count == 3, parts[2].hasPrefix("HTTP/1.") else {
            throw ParseError.malformed
        }
        var headers: [String: String] = [:]
        for line in lines.dropFirst() {
            guard let colonIx = line.firstIndex(of: ":") else { continue }
            let key = String(line[..<colonIx]).trimmingCharacters(in: .whitespaces)
            let val = String(line[line.index(after: colonIx)...]).trimmingCharacters(in: .whitespaces)
            headers[key] = val
        }
        let bodyStart = headerEnd + 4
        let contentLength = Int(headers["Content-Length"] ?? "0") ?? 0
        guard bytes.count >= bodyStart + contentLength else {
            throw ParseError.needMoreData
        }
        let body = Array(bytes[bodyStart..<bodyStart + contentLength])
        return Request(method: parts[0], path: parts[1], headers: headers, body: body)
    }

    private static func findSequence(_ needle: [UInt8], in haystack: [UInt8]) -> Int? {
        guard needle.count <= haystack.count else { return nil }
        for i in 0...(haystack.count - needle.count) {
            if Array(haystack[i..<i + needle.count]) == needle { return i }
        }
        return nil
    }
}
```

```swift
extension HTTPCodec {
    static func writeResponse(status: Int, contentType: String, body: [UInt8]) -> [UInt8] {
        let reason: String
        switch status {
        case 200: reason = "OK"
        case 400: reason = "Bad Request"
        case 404: reason = "Not Found"
        case 408: reason = "Request Timeout"
        case 500: reason = "Internal Server Error"
        default:  reason = "OK"
        }
        var head = "HTTP/1.1 \(status) \(reason)\r\n"
        head += "Content-Type: \(contentType)\r\n"
        head += "Content-Length: \(body.count)\r\n"
        head += "Connection: close\r\n"
        head += "\r\n"
        return Array(head.utf8) + body
    }
}
```

```swift
// Sources/Nudge/PromptQueue.swift
import Foundation

/// FIFO queue of pending Prompts. Each enqueue suspends until the queue is
/// resolved with a Decision. Multiple producers (HTTP server) feed the queue;
/// a single consumer (the menu bar UI) resolves the head.
actor PromptQueue {
    private struct Pending {
        let prompt: Prompt
        let continuation: CheckedContinuation<Decision, Error>
    }

    private var pending: [Pending] = []

    /// Listener notified when the head of the queue changes
    /// (new prompt arrives at empty queue, or head is resolved/removed).
    /// Set by MenuBarController.
    var onHeadChange: ((Prompt?, Int) -> Void)?

    enum QueueError: Error {
        case timedOut
    }

    /// Enqueue a prompt and suspend until it is resolved.
    func enqueue(_ prompt: Prompt) async throws -> Decision {
        return try await withCheckedThrowingContinuation { cont in
            let item = Pending(prompt: prompt, continuation: cont)
            pending.append(item)
            if pending.count == 1 { notifyHead() }
        }
    }

    /// Resolve the head of the queue. No-op if empty.
    func resolveHead(with decision: Decision) {
        guard !pending.isEmpty else { return }
        let head = pending.removeFirst()
        head.continuation.resume(returning: decision)
        notifyHead()
    }

    /// Time out the head of the queue. No-op if empty.
    func timeoutHead() {
        guard !pending.isEmpty else { return }
        let head = pending.removeFirst()
        head.continuation.resume(throwing: QueueError.timedOut)
        notifyHead()
    }

    func setOnHeadChange(_ cb: @escaping (Prompt?, Int) -> Void) {
        onHeadChange = cb
        notifyHead()
    }

    private func notifyHead() {
        onHeadChange?(pending.first?.prompt, pending.count)
    }
}
```

```swift
extension PromptQueue {
    /// Enqueues with a timeout. If `seconds` elapses before the head is resolved,
    /// throws `.timedOut` and removes the prompt from the queue.
    func enqueueWithTimeout(_ prompt: Prompt, seconds: TimeInterval) async throws -> Decision {
        try await withThrowingTaskGroup(of: Decision.self) { group in
            group.addTask { try await self.enqueue(prompt) }
            group.addTask {
                try await Task.sleep(nanoseconds: UInt64(seconds * 1_000_000_000))
                await self.removePrompt(id: prompt.id)
                throw QueueError.timedOut
            }
            let first = try await group.next()!
            group.cancelAll()
            return first
        }
    }

    fileprivate func removePrompt(id: String) {
        if let idx = pending.firstIndex(where: { $0.prompt.id == id }) {
            let removed = pending.remove(at: idx)
            removed.continuation.resume(throwing: QueueError.timedOut)
            notifyHead()
        }
    }
}
```

```swift
// Sources/Nudge/PromptServer.swift
import Foundation
import Network

actor PromptServer {
    private let queue: PromptQueue
    private let requestedPort: NWEndpoint.Port
    private var listener: NWListener?
    private(set) var boundPort: UInt16 = 0
    private let timeoutSeconds: TimeInterval

    init(queue: PromptQueue, port: UInt16, timeoutSeconds: TimeInterval = 300) {
        self.queue = queue
        self.requestedPort = port == 0 ? .any : NWEndpoint.Port(rawValue: port)!
        self.timeoutSeconds = timeoutSeconds
    }

    func start() async throws {
        let params = NWParameters.tcp
        let listener = try NWListener(using: params, on: requestedPort)
        self.listener = listener

        listener.newConnectionHandler = { [weak self] conn in
            Task { await self?.handle(connection: conn) }
        }

        let started = AsyncStream<UInt16> { cont in
            listener.stateUpdateHandler = { state in
                if case .ready = state, let port = listener.port {
                    cont.yield(port.rawValue)
                    cont.finish()
                } else if case .failed(let err) = state {
                    cont.finish()
                    print("PromptServer failed: \(err)")
                }
            }
            listener.start(queue: .global())
        }
        for await p in started { boundPort = p; break }
    }

    func stop() {
        listener?.cancel()
        listener = nil
    }

    private func handle(connection: NWConnection) async {
        connection.start(queue: .global())
        var buf: [UInt8] = []
        while true {
            let chunk: [UInt8] = await withCheckedContinuation { cont in
                connection.receive(minimumIncompleteLength: 1, maximumLength: 65_536) { data, _, _, _ in
                    cont.resume(returning: data.map { Array($0) } ?? [])
                }
            }
            if chunk.isEmpty { break }
            buf.append(contentsOf: chunk)
            do {
                let req = try HTTPCodec.parseRequest(buf)
                await respond(to: req, on: connection)
                connection.cancel()
                return
            } catch HTTPCodec.ParseError.needMoreData {
                continue
            } catch {
                let resp = HTTPCodec.writeResponse(status: 400, contentType: "text/plain", body: Array("bad request".utf8))
                connection.send(content: Data(resp), completion: .contentProcessed { _ in connection.cancel() })
                return
            }
        }
        connection.cancel()
    }

    private func respond(to req: HTTPCodec.Request, on conn: NWConnection) async {
        guard req.method == "POST", req.path == "/prompt" else {
            let resp = HTTPCodec.writeResponse(status: 404, contentType: "text/plain", body: Array("not found".utf8))
            conn.send(content: Data(resp), completion: .contentProcessed { _ in })
            return
        }
        let prompt: Prompt
        do {
            prompt = try JSONDecoder().decode(Prompt.self, from: Data(req.body))
        } catch {
            let resp = HTTPCodec.writeResponse(status: 400, contentType: "text/plain", body: Array("bad json".utf8))
            conn.send(content: Data(resp), completion: .contentProcessed { _ in })
            return
        }
        do {
            let decision = try await queue.enqueueWithTimeout(prompt, seconds: timeoutSeconds)
            let body = try JSONEncoder().encode(DecisionResponse(decision: decision))
            let resp = HTTPCodec.writeResponse(status: 200, contentType: "application/json", body: Array(body))
            conn.send(content: Data(resp), completion: .contentProcessed { _ in })
        } catch {
            let resp = HTTPCodec.writeResponse(status: 408, contentType: "text/plain", body: Array("timeout".utf8))
            conn.send(content: Data(resp), completion: .contentProcessed { _ in })
        }
    }
}
```

```swift
import Foundation

enum PortFile {
    static var defaultURL: URL {
        let home = FileManager.default.homeDirectoryForCurrentUser
        return home.appendingPathComponent(".config/nudge/port")
    }

    enum FileError: Error {
        case missing
        case malformed
    }

    static func write(port: UInt16, to url: URL = defaultURL) throws {
        try FileManager.default.createDirectory(at: url.deletingLastPathComponent(), withIntermediateDirectories: true)
        try "\(port)\n".write(to: url, atomically: true, encoding: .utf8)
    }

    static func read(from url: URL = defaultURL) throws -> UInt16 {
        guard FileManager.default.fileExists(atPath: url.path) else { throw FileError.missing }
        let raw = try String(contentsOf: url, encoding: .utf8).trimmingCharacters(in: .whitespacesAndNewlines)
        guard let port = UInt16(raw) else { throw FileError.malformed }
        return port
    }
```

### Task 21: Global Enter/Escape while popover is open

The `keyboardShortcut(.defaultAction)` and `.cancelAction` modifiers in PopoverView already handle Enter/Escape when the popover has key focus. We need it to also work when the popover is shown but the user's focus is in another app (their browser, etc.).

- [ ] **Step 1: Add a global event monitor active only while the popover is open**

Add to `MenuBarController`:

```swift
private var keyMonitor: Any?

private func startKeyMonitor() {
    stopKeyMonitor()
    keyMonitor = NSEvent.addGlobalMonitorForEvents(matching: .keyDown) { [weak self] event in
        guard self?.popover.isShown == true else { return }
        if event.keyCode == 36 || event.keyCode == 76 { // Return / Enter
            DispatchQueue.main.async { self?.resolve(.allow) }
        } else if event.keyCode == 53 { // Escape
            DispatchQueue.main.async { self?.resolve(.deny) }
        }
    }
}

private func stopKeyMonitor() {
    if let m = keyMonitor { NSEvent.removeMonitor(m); keyMonitor = nil }
}
```

```bash
#!/usr/bin/env bash
# scripts/install-hook.sh — merge Nudge's PreToolUse hook into ~/.claude/settings.json
set -euo pipefail

SETTINGS="$HOME/.claude/settings.json"
HOOK_CMD="/Applications/Nudge.app/Contents/MacOS/nudge-hook"
BACKUP="$SETTINGS.bak.$(date +%s)"

if [[ ! -f "$SETTINGS" ]]; then
  echo "{}" > "$SETTINGS"
fi

# Validate current JSON.
jq -e . "$SETTINGS" > /dev/null

# Backup.
cp "$SETTINGS" "$BACKUP"
echo "Backed up settings to $BACKUP"

# Merge: add a PreToolUse entry under hooks. If one already exists, append our
# entry (don't overwrite). Use jq to preserve formatting and other keys.
jq --arg cmd "$HOOK_CMD" '
  .hooks //= {} |
  .hooks.PreToolUse //= [] |
  .hooks.PreToolUse += [{
    "if": "Bash(*)",
    "hooks": [{ "type": "command", "command": $cmd }]
  }]
' "$SETTINGS" > "$SETTINGS.tmp"

# Validate the merged file before replacing.
jq -e . "$SETTINGS.tmp" > /dev/null
mv "$SETTINGS.tmp" "$SETTINGS"

echo "Installed Nudge hook in $SETTINGS"
```

```bash
#!/usr/bin/env bash
# scripts/uninstall-hook.sh — remove Nudge's hook from ~/.claude/settings.json
set -euo pipefail

SETTINGS="$HOME/.claude/settings.json"
HOOK_CMD="/Applications/Nudge.app/Contents/MacOS/nudge-hook"
BACKUP="$SETTINGS.bak.$(date +%s)"

[[ -f "$SETTINGS" ]] || { echo "no settings file, nothing to remove"; exit 0; }

  else . end
' "$SETTINGS" > "$SETTINGS.tmp"
```

- [ ] **Step 1: Write the README**

````markdown
# Nudge

A macOS menu bar app that surfaces Claude Code permission prompts as a clickable popover, so you can approve from anywhere without alt-tabbing to the terminal.

## How it works

1. Claude Code is about to run a Bash command that auto mode flags as needing user approval.
2. Claude's `PreToolUse` hook fires `nudge-hook`, which POSTs the prompt to Nudge's local HTTP server.
3. Nudge's menu bar icon flips to a soft red pulse and a popover auto-opens showing the command.
4. You click **Allow** (or hit Enter) — even from another app, no focus shift required.
5. Claude Code receives the decision and proceeds.

If Nudge is broken in any way (not running, crashed, settings file missing), Claude Code falls back to its normal terminal prompt. Nudge can never block you.

## Requirements

- macOS 14 (Sonoma) or later
- Swift 5.10 toolchain (`xcode-select --install` is enough)
- `jq` (`brew install jq`) — used by the install script
- Claude Code with auto mode enabled

## Install

```bash
git clone <this repo>
cd nudge
make install
open -ga Nudge
```

The first launch will:

1. Prompt for **Accessibility** permission (System Settings → Privacy & Security → Accessibility) — needed for global Enter/Escape shortcuts while the popover is open.
2. Show a small filled circle in your menu bar.

## Uninstall

```bash
make uninstall
```

Removes `/Applications/Nudge.app` and unsplices the hook from `~/.claude/settings.json` (a backup is created at `~/.claude/settings.json.bak.<timestamp>` first).

## Troubleshooting

- **Popover doesn't auto-open:** confirm Nudge is running (`pgrep Nudge`). Confirm the port file exists (`cat ~/.config/nudge/port`).
- **Claude Code still shows its terminal prompt:** the hook may have failed silently. Check `claude --debug` output for hook errors. Also verify `jq -e . ~/.claude/settings.json` parses cleanly.
- **Enter/Escape don't work from other apps:** Accessibility permission isn't granted. Open System Settings → Privacy & Security → Accessibility and toggle Nudge on.
- **"Nudge can't be opened because it's from an unidentified developer":** Run `xattr -dr com.apple.quarantine /Applications/Nudge.app`. The installer does this automatically; if you copied the app manually, you may need to re-run it.

## Architecture

See [`docs/superpowers/specs/2026-04-28-nudge-design.md`](docs/superpowers/specs/2026-04-28-nudge-design.md).
````
